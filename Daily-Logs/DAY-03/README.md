## Key Learnings
- Assignment and simple if conditions generally result in O(1)
- Single loops contribute to O(n)
- Nested loops contribute to O(n²)
- Loops with doubling control variables result in O(log₂n)
- Space Complexity depends on auxiliary memory usage
- Difference between linear and logarithmic growth of algorithms
